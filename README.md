Docker Node Yarn Chrome Xvfb, etc.
==================================

Docker image which has installed the following:
- Node
- Yarn
- Chrome
- Xvfb
- JDK 10
- AWS CLI
- jq
- build-essential
- dependencies for node-canvas

Also see [Docker Hub](https://hub.docker.com/r/canvadev/ci-docker-node-yarn-chrome-xvfb/).
